# Credit Risk Prediction

## Project Overview
This project predicts credit risk using Machine Learning techniques. The model helps financial institutions assess whether a loan applicant is likely to be a low-risk or high-risk borrower.

## Technologies Used
- Python
- Pandas
- NumPy
- Scikit-Learn
- Jupyter Notebook

## Machine Learning Algorithm
- Random Forest Classifier

## Objective
To analyze applicant information and predict credit risk for better lending decisions.

## Features
- Age
- Income
- Employment Length
- Home Ownership
- Loan Amount
- Interest Rate
- Loan Intent
- Credit History Length
- And Other Financial Attributes

## Target Variable
- Loan Status / Credit Risk

## Workflow
1. Data Collection
2. Data Preprocessing
3. Feature Encoding
4. Train-Test Split
5. Model Training
6. Prediction
7. Evaluation

## Result
The model predicts credit risk based on applicant information and supports data-driven loan approval decisions.
